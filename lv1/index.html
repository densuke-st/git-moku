レベル1: ローカルコミット¶

目標は以下のあたり

  • gitのリポジトリをローカルで作れます

  • 他の人のリポジトリを引き出せます

  • とりあえずコミットできます

  • 他の人のソースを取得する
    • 取得元を確認する
    • cloneを行う
      • コマンドラインで行う
      • vscodeで行う
      • Eclipseで行う
      • Mavenプロジェクトなどの場合
    • zipと何が違うの?
  • 新規にリポジトリを作る
    • vscodeにおけるinit
      • git で作成したディレクトリを開く
      • vscodeの中で初期化する
    • Eclipseの中で初期化する
      • 「プロジェクトの親フォルダー内のリポジトリーを使用または作成」 のチェックを使う
      • 別の場所に用意されるリポジトリ専用エリアを使用する
  • リポジトリ操作
    • ファイルの登録
    • コミット
      • vscodeでの操作
    • ファイルを編集してみる
    • 今度は自分でやってみよう
