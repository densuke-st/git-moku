.. _level1:

=========================================
レベル1: ローカルコミット
=========================================


目標は以下のあたり

- gitのリポジトリをローカルで作れます
- 他の人のリポジトリを引き出せます
- とりあえずコミットできます

ここまでができると、ローカルのmain(旧master)ブランチに対し、最低限ファイルを作ってコミット(登録)が行えるようになります。
ただし、まだ一人での作業にしか使えません。
